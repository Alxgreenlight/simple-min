Версии солвера с возможностью замера времени выполнения отдельных частей кода
Для просмотра времени после расчеита функции, public метод
virtual void debug_time(std::ofstream & fp)

Реализован подсчет времени (всего потрачено при решениии задачи)
1. gridtime на расчет значений функции в узлах сетки
2. esttime на расчет оценок константы Липшица
3. minsearch на нахождение верхней оценки
